## Overview

BulkFileChanger gives Windows users a faster way to handle file metadata changes in groups. Rather than modifying attributes or timestamps file by file, it streamlines the process so the same edits can be applied across many items with less manual work.

It is intended for routine maintenance, organizing files, and repeating the same edits on Windows 10 and Windows 11. The project offers both portable and installer-based distribution, so it can adapt to different ways of deploying the tool while keeping the emphasis on bulk file attribute editing.
